# 🚀 Overview

This repository contains the code and testing evidence for **PELEC305 Quiz 2**.  
The project demonstrates how to create a secure Django form that allows users to register for an event while validating user input before saving it to the database.



# 🧾 Validation Rules

The form validates the following requirements:

| Field | Validation Rule |
|---|---|
| Full Name | Must be at least 5 characters |
| Email | Must end with `@gmail.com` |
| Age | Must be 18 and above |
| Password | Must be at least 8 characters |
